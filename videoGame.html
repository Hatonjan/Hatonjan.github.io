The Nightfall Adventure

The Nightfall Adventure is a 2D video game developed using JavaScript and the P5.js library. All visuals and interactions were created by applying programming principles, avoiding the use of external engines or assets. The game features a world where the player must navigate through a dark forest to find all the monsters that threaten the peaceful lives of the villagers.
